## 简介

本作品硬件上使用ESP32作为主控，软件上使用platformio编写程序，完成了远程开机卡的设计制作。可以在局域网内完成远程开机任务。

## 硬件

- ESP32-WROVER
- TFT显示屏
- AHT温度传感器


## 软件

- PlatformIO

## 功能

- 远程开机
- 网络控制台
- 温度显示
- 屏幕显示（可以显示中文，感谢https://github.com/StarCompute/tftziku

## 使用方法

下载文件后使用platformio打开（第一次打开会比较慢）
